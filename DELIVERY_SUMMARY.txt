═══════════════════════════════════════════════════════
DESIGN PRINCIPLES (from Anthropic's Agent Skills spec)
═══════════════════════════════════════════════════════

- Progressive disclosure: frontmatter name+description loads at
  startup; body loads on demand; bundled files load only when
  referenced.
- Short bodies (60-200 lines each, 58-176 in practice).
- Cross-references via a "See also" block at the bottom of each
  skill. All 26 cross-refs resolve to real siblings (validated).
- Descriptions are 236-438 chars each — specific enough to trigger
  automatically, short enough to scan at load time.

HOW TO USE IN WINDSURF (TL;DR)
═══════════════════════════════════════════════════════

    git clone https://github.com/tabgab/opp_repl-skill.git ~/opp_repl-skill
    mkdir -p ~/.codeium/windsurf/skills
    for dir in ~/opp_repl-skill/opp-repl-*; do
        ln -sfn "$dir" ~/.codeium/windsurf/skills/
    done

Restart Windsurf; @-mention any skill in the Cascade panel (e.g.
`@opp-repl-overview`) or let Cascade auto-invoke based on prompt.
Full walkthrough (UI option, workspace-scoped option, Windows
PowerShell equivalent, MCP hookup) in WINDSURF.md.
